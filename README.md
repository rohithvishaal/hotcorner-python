# Hotcorner - Python
Implementation of Hotcorner feature that you see in Ubuntu and other Linux Distros

## Installation - Linux
`pip3 install pyautogui`

## Installation - Windows
`pip install pyautogui`

## How to Run - Linux
`python3 hotcorner.py`

## To Stop - Linux
`pkill hotcorner`

## How to Run - Windows 
`python hotcorner.py`

## To Stop - Windows
`taskkill /IM "hotcorner.exe" /F`
